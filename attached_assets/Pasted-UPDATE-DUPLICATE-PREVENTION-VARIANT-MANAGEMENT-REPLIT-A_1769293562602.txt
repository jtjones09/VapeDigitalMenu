Update the duplicate search API response:
File: /server/routes/products.ts
In the POST /api/products/search-duplicates endpoint:
CURRENT response structure:

  matches: [
    { id, productName, brandName, productType, similarity, variantCount }
  ]
}
NEW response structure - ADD inShopMenu field:
typescript{
  matches: [
    { 
      id, 
      productName, 
      brandName, 
      productType, 
      similarity, 
      variantCount,
      inShopMenu: boolean  // ← ADD THIS
    }
  ]
}
Implementation:
For each matched product, check if it exists in shop_products table for the current shop:
sql-- For each matched product, check if it's in shop's menu
SELECT 
  p.*,
  COALESCE(b.brand_name, p.custom_brand_name) as brand_name,
  SIMILARITY(p.product_name, $searchTerm) as similarity,
  COUNT(DISTINCT pv.id) as variant_count,
  CASE 
    WHEN sp.product_id IS NOT NULL THEN true 
    ELSE false 
  END as in_shop_menu
FROM products p
LEFT JOIN brands b ON p.brand_id = b.id
LEFT JOIN product_variants pv ON pv.product_id = p.id
LEFT JOIN shop_products sp ON sp.product_id = p.id AND sp.shop_id = $currentShopId
WHERE p.is_custom = false
  AND SIMILARITY(p.product_name, $searchTerm) > 0.6
GROUP BY p.id, b.brand_name, sp.product_id
ORDER BY similarity DESC
LIMIT 5;
```

**Key point:** The LEFT JOIN with `shop_products` tells us if this product is already in the shop's menu.
```

typescriptconst handleUseProduct = async (productId, match) => {
  // Check if already in menu (from match.inShopMenu flag)
  if (match.inShopMenu) {
    toast({
      title: "Already Added",
      description: "This product is already in your menu",
      variant: "default"
    });
    return;

  try {

      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        shopId: currentShopId,
        productId: productId,
        isActive: true,
        displayOrder: 0
      })
    });
    
    toast({
      title: "Product Added!",
      description: `${match.productName} has been added to your menu`,
    });
    
    // Refresh matches to update "In Menu" status
    refetchMatches();
    
    // Optionally redirect to products page
    // navigate('/admin/products');
    
  } catch (error) {
    toast({
      title: "Error",
      description: error.message || "Failed to add product",
      variant: "destructive"
    });
  }
};
